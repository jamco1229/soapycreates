# Soapy

## Commands

| Command            | Action                                      |
| :----------------- | :------------------------------------------ |
| `pnpm install`     | Install dependencies                         |
| `pnpm run dev`     | Start local dev server at `localhost:4321`   |
| `pnpm run build`   | Build the production site to `./dist/`       |
| `pnpm run preview` | Preview the build locally                    |
